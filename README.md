Este repositorio es usado para la entrega del parcial numero III. 

Con la implementación del Sistema de Gestión de Inventario, se espera transformar este proceso manual en uno automatizado, preciso y accesible,
que brinde a las empresas una herramienta tecnológica capaz de agilizar las operaciones internas,
mejorar el control de productos y optimizar la utilización de los recursos. El cambio permitirá que los encargados tengan una visión clara y en tiempo real del estado del inventario,
garantizando una mejor organización, reducción de errores y una toma de decisiones más informada.
